### Common WPF dev mistakes:

❌ Using Items and ItemsSource together

❌ Forgetting to set DataContext

❌ Using List<T> instead of ObservableCollection<T>

❌ Expecting UI to update without INotifyPropertyChanged

❌ Blocking UI thread with long operations

❌ Not using async/await for async calls

❌ Hardcoding strings instead of using resources

❌ Ignoring memory leaks from event handlers

❌ Overusing code-behind instead of MVVM pattern

❌ Not utilizing data templates for complex data

❌ Neglecting to handle exceptions in async methods
